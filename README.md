# vue_SwipeModal

## Description
スワイプして閉じることのできるモーダルウィンドウです。（通称:
 Swipeable Bottom Sheet）

## DEMO
https://vue-swipe-modal.vercel.app/

## Requirement
| name | version |
| --- | --- |
| nuxt.js | 2.15.3 |
| core-js | 3.9.1 |

## Getting Started
```
<template>
  <div>
    <Swipemodal
      height="50vh"
      width="50%"
      radius="20px"
      notip
    >
      <h1>contents</h1>
    </Swipemodal>
  </div>
</template>
```
